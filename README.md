## Installation
1. Download and place in your `resources` directory
2. Add `start esx_goggles` to your server.cfg
3. Ensure `nightvision_goggles` item exists in your database

## Controls
- `Z` - Toggle Goggles On/Off
- `J` - Cycle through vision modes
